add_tooltip(*, labels=None, groups=None, tooltip_x_shift=0, tooltip_y_shift=0, hover_nearest=False, on=None, ax=None) ¶

Add a tooltip to the interactive plot. You can set either just labels, just groups, both or none.

Parameters:

Name Type Description Default
labels list | tuple | ndarray | SeriesT | None

An iterable containing the labels for the tooltip. It corresponds to the text that will appear on hover.

 None
groups list | tuple | ndarray | SeriesT | None

An iterable containing the group for tooltip. It corresponds to how to 'group' the tooltip. The easiest way to understand this argument is to check the examples below. Also note that the use of this argument is required to 'connect' the legend with plot elements.

 None
tooltip_x_shift int

Number of pixels to shift the tooltip from the cursor, on the x axis.

 0
tooltip_y_shift int

Number of pixels to shift the tooltip from the cursor, on the y axis.

 0
hover_nearest bool

When True, hover the nearest plot element.

 False
on str | list[str] | None

Which plot elements to apply interactivity to. Can be a single element type or a list. Valid values are "point", "line", "bar", "area" (plurals like "points" also accepted). If None (default), applies to all element types.

 None
ax Axes | None

A matplotlib Axes. If None (default), uses first Axes.

 None

Returns:

Name Type Description
self PlotJS

Returns the instance to allow method chaining.

Examples:

PlotJS(...).add_tooltip(
    labels=["S&P500", "CAC40", "Sunflower"],
)

PlotJS(...).add_tooltip(
    labels=["S&P500", "CAC40", "Sunflower"],
    columns=["S&P500", "CAC40", "Sunflower"],
)

PlotJS(...).add_tooltip(
    labels=["S&P500", "CAC40", "Sunflower"],
    hover_nearest=True,
)

PlotJS(...).add_tooltip(
    labels=["S&P500", "CAC40", "Sunflower"],
    on="point",  # only apply hover to points
)

PlotJS(...).add_tooltip(
    labels=["S&P500", "CAC40", "Sunflower"],
    on=["point", "line"],  # apply hover to points and lines only
)

as_html() ¶

Retrieve the interactive plot as an HTML string. This can be useful to display the plot in environment such as marimo, or do advanced customization.

Returns:

Type Description
str

A string with all the HTML of the plot.

Examples:

import marimo as mo
from plotjs import PlotJS, data

df = data.load_iris()

html_plot = (
    PlotJS(fig=fig)
    .add_tooltip(labels=df["species"])
    .as_html()
)

# display in marimo
mo.iframe(html_plot)

show() ¶

Open the HTML file in the default browser. If the file hasn't been saved yet, it will be saved to a temporary file.

Returns:

Name Type Description
self PlotJS

Returns the instance to allow method chaining.

Examples:

PlotJS(fig).save("output.html").show()

# Open without explicitly saving (uses temp file)
PlotJS(fig).show()
